# Hiperthi — Sincretism

Este projeto explora a ideia de uma linguagem sincrética, onde múltiplos nomes podem designar o mesmo conceito, função ou ação. A proposta é permitir flexibilidade, facilitando a expressão do mesmo comando de diferentes formas.

## Ideia inicial

- Mostrar exemplos em Python de funções com nomes diferentes realizando a mesma operação.
- Refletir sobre como linguagens e tecnologias possuem diferentes “palavras” para os mesmos significados.

## Exemplos

Veja em `sincretico.py` como “somar”, “adicionar”, “plus” ou “sum” podem ser vistas como equivalentes.

1. Clone o repositório:
   ```sh
   git clone https://github.com/Superthi/Hiperthi.git
   ```
